# AI Reality Check

A simulated logical claim analyzer built with React, TypeScript, and Vite.

## Features

- **Viral Claim Analysis**: Input text for logical evaluation.
- **Simulated Logic Engine**: Returns deterministic verdicts based on keyword analysis.
- **Premium UI**: Glassmorphism design with deep space aesthetics.
- **Privacy-First**: All analysis happens client-side (mock).

## Tech Stack

- React + TypeScript
- Vite
- Vanilla CSS (Glassmorphism)
- Lucide React Icons

## Getting Started

1. `npm install`
2. `npm run dev`
